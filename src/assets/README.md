# Assets Folder

Place your custom assets here:

## Structure

- `/images/` - Photo memories, backgrounds
- `/music/` - Background music files (mp3)
- `/gifs/` - Animated GIFs

## Usage

Import assets in your components:

```tsx
import myImage from "@/assets/images/my-photo.jpg";
```

## Notes

- Recommended image formats: jpg, png, webp
- Recommended music formats: mp3, ogg
- Keep file sizes optimized for web
